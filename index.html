Project: Diagnosis of Tomato Leaf Diseases Using K-Means Clustering and Machine Learning

Objective:

Develop an intelligent machine vision system for diagnosing and classifying tomato leaf diseases through advanced image processing and machine learning techniques.

Description:

This project involves creating a system that: Diagnoses and classifies tomato leaf diseases using a novel image processing algorithm. Utilizes K-means clustering to identify disease areas. The proposed method outperformed deep learning models like CNN and GoogleNet in accuracy while significantly reducing processing time.
